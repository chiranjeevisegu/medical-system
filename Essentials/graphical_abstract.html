BioClinicalBERT Tokenization
WordPiece tokenizer processes summaries at max 512 tokens. CLS embedding (768-dim) extracted from the final transformer layer.
768-dim CLS vector → Linear Head → Softmax
②   Multi-Agent Reasoning (FLAN-T5)
Classification Layer
BioClinicalBERT + Linear Head maps unstructured discharge text to one of five structured disease signals. Supervised with AdamW (lr=2e⁻⁵), evaluated by Precision / Recall / F1.
01
Risk Agent
Assigns Low / Medium / High risk with structured clinical rationale grounded in report indicators.
02
Explainability Agent
Translates clinical findings into FKGL ≤8 patient-friendly language using analogies and plain vocabulary.
03
Recommendation Agent
Generates 3–5 safe, non-diagnostic next steps emphasising follow-up care and symptom monitoring.
04
Justification Agent
Explains recommendation rationale, confidence level, and explicit limitations for interpretability.
05
Verification Agent
Detects contradictions and unsafe language across all agent outputs. Triggers re-generation if flagged.
